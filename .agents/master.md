# Portfolio Redesign Master Guide

This document serves as a comprehensive guide for the AI coding agent assisting in the redesign of Youssef Eltalyawy's portfolio website. It outlines the skills, tools, and best practices to be used throughout the project.
This file serves as the definitive source of truth, configuration instructions, and engineering constraints for all development phases of the Talyawy portfolio website redesign.

---

## 📁 Project Overview

- **Project Name**: talyawy.dev Portfolio Redesign
- **Objective**: Create a unique, pixel-perfect, and interactive portfolio website that showcases Youssef's work, skills, and personality.
- **Tech Stack**:
  - **Framework**: Next.js (App Router)
  - **Styling**: Tailwind CSS v4
  - **Animations**: GSAP + ScrollTrigger
  - **3D/Shaders**: Three.js + GLSL
  - **Deployment**: Vercel
